# Jogo De Adivinhação

![](https://imgur.com/gc0mB3y.gif)

## Introdução

 O computador pensará em um número, e você, jogador, precisará adivinhá-lo. A cada erro, a máquina lhe dirá se o número chutado foi maior ou menor do que o pensado.

Você também poderá escolher o nível de dificuldade do jogo, e isso lhe dará mais ou menos oportunidades de chutar um número.

## Funcionalidades

- Uma estrutura básica do jogo capaz de capturar a entrada do jogador
- Gerar um número aleatório com Random
- Implementa a lógica de verificação do palpite
- Múltiplas tentativas e dificuldade
